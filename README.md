# CV Builder

A browser-based CV builder application built with React and CSS. It lets users fill in CV sections and display the saved data in a preview.

Live Demo: https://cv-builder-darius.netlify.app/

## Technologies Used

React: Component-based structure for the form sections and CV preview.

CSS3: Layout and styling for the editor and preview.

Vite: Development server and production build tool for the React application.

React Icons: Icons used in the form sections and preview.

## Features

Editable Forms: Users can fill in general information, education, and work experience details.

CV Preview: The preview section displays the saved resume data in a CV layout.

Collapsible Sections: Form sections can be opened and closed while editing.

## Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```
